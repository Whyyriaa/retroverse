Diseño del Registro Principal

Registro principal: RegVintage

Campo Tipo Descripción
Cprod Entero Código único del producto. Clave primaria.
Nombprod AN(50) Nombre del producto.
Categ AN(30) Categoría del producto.
Marca AN(30) Marca del producto.
Precio Real Precio de venta.
Stock Entero Cantidad disponible.
Estado AN(20) Estado del producto.
Vendedor AN(50) Usuario o nombre del vendedor.

Clave del registro: Cprod. Se eligió este campo porque identifica de manera única cada producto dentro de RetroVerse. Aunque dos productos puedan tener nombres parecidos, cada uno tendrá un código distinto, lo que permite buscar, modificar o eliminar registros sin confusiones.

Ayuda / Tutorial

  1. Explorar productos: ingresá al catálogo y mirá las tarjetas disponibles.
  2. Filtrar productos: usá los botones Anime, Videojuegos o Colección.
  3. Ver detalles: presioná “Ver detalle” para abrir información ampliada del producto.
  4. Agregar al carrito: presioná “Agregar al carrito” para sumar un producto a la compra.
  5. Revisar carrito: entrá a la sección Carrito para ver productos agregados y total.
  6. Ver registro: en Registro se muestra el diseño del registro principal RegVintage.
